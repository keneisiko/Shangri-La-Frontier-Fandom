# Инструкция по установке и запуску

## Быстрый старт

1. **Установите Python 3.8+** (если еще не установлен)

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Примените миграции базы данных:**
```bash
python manage.py makemigrations
python manage.py migrate
```

4. **Создайте суперпользователя (опционально, для доступа к админ-панели):**
```bash
python manage.py createsuperuser
```

5. **Запустите сервер разработки:**
```bash
python manage.py runserver
```

6. **Откройте браузер и перейдите по адресу:**
```
http://127.0.0.1:8000/
```

## Первые шаги

1. Зарегистрируйте новый аккаунт
2. Войдите в систему
3. Создайте свой первый пост, обсуждение или фанфик!

## Админ-панель

Для доступа к админ-панели Django перейдите по адресу:
```
http://127.0.0.1:8000/admin/
```

Используйте учетные данные суперпользователя, созданного ранее.

## Структура проекта

- `fandom/` - основное приложение
- `fandom_project/` - настройки проекта
- `media/` - загруженные пользователями файлы (изображения)
- `static/` - статические файлы (CSS, JS)
- `db.sqlite3` - база данных (создается автоматически)

## Решение проблем

Если возникают ошибки:
1. Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
2. Проверьте, что миграции применены: `python manage.py migrate`
3. Убедитесь, что директории `media/` и `static/` существуют
