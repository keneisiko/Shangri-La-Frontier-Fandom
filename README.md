# Фандом Рубеж Шангрила

Веб-сайт на Django для сообщества поклонников аниме "Рубеж Шангрила".

## Возможности

- ✅ Регистрация и вход пользователей
- ✅ Профили пользователей с аватарами
- ✅ Создание и редактирование постов с изображениями
- ✅ Обсуждения с комментариями
- ✅ Фанфики с фильтрацией по жанрам и рейтингам
- ✅ Система лайков для постов и фанфиков
- ✅ Современный и красивый UI

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Примените миграции:
```bash
python manage.py makemigrations
python manage.py migrate
```

3. Создайте суперпользователя (опционально):
```bash
python manage.py createsuperuser
```

4. Запустите сервер разработки:
```bash
python manage.py runserver
```

5. Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/

## Структура проекта

- `fandom_project/` - настройки проекта Django
- `fandom/` - основное приложение
  - `models.py` - модели данных (Post, Discussion, Fanfic, Profile, Comment)
  - `views.py` - представления (views)
  - `forms.py` - формы
  - `urls.py` - URL маршруты
  - `templates/` - HTML шаблоны
- `media/` - загруженные файлы (изображения)
- `static/` - статические файлы (CSS, JS)

## Использование

1. Зарегистрируйтесь или войдите в систему
2. Создайте пост, обсуждение или фанфик
3. Общайтесь с другими пользователями
4. Читайте и пишите фанфики по вселенной "Рубеж Шангрила"

## Технологии

- Django 4.2
- Bootstrap 5
- SQLite (база данных)
- Pillow (обработка изображений)
